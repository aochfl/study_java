Web Application Server 필수 -> WAS
Tomcat 9.0 버전

내 코드를 서버에 전달해서 실행시켜라
-> 웹브라우저를 통해서 주소로 서버에서 실행된 페이지에 접근
-> localhost, 127.0.0.1, ip+port 로 접근
-> kr nick - 한국에 들어오는 모든 아이피를 관리하는 단체
-> krnick -> kt,up, sk, -> 기본적으로 ip만 가지고 국가를 알 수 있다. 대충의 지역도 알 수 있다.

port : ip를 집주소라고 한다면 port는 집 안에 방번호다.
=> ip는 대부분 String, port는 대부분 int로 취급됨

80 : http - 생략가능
21 - ftp
22 - sftp
3306 - mysql
5560 - oracle 10g
8080 - http 테스트용 포트

port 는 변경가능하다. 다만, 대부분의 프로그램들이 묵시적으로 사용하는 포트번호가 있으므로 
혼자바꾸면 혼자만 돌아간다.

서버만 실행되면 모든 사람들이 접속 가능하다.
하지만 서버를 계속 켜두는 데 돈이 많이든다.
그런 이유로 그런것만 해주는 서비스가 있다. - 호스팅

호스팅 => 어떤 서버, 어떤 언어, 어떤 디비에 따라서 가격차이가 있고
하루에 최대로 사용할 수 있는 접속 용량이 있다.
